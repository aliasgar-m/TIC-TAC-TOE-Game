Information Page

Rules of the Game:

1. The game is played on a grid that's 3 squares by 3 squares.

2. Each player takes turns to mark an empty square.

3. The first player to get 3 marks in a row (up, down, across, or diagonally) is the winner.

4. When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.


Tips:

1. Figure out how to get three marks in a row.

2. Think ahead.

3. Carefully observe the opponent's moves
