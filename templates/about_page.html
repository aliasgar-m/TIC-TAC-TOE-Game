A Tic Tac Toe game based on Reinforcement Learning.

Author: Aliasgar Merchant

Email: aliasgarm2000@gmail.com

About Game: The AI is trained using Iterative Q-Learning. The computer has trained itself over 500,000 games thereby reaching the optimal policy using a reward based approach. Rewards given to the computer include:

   1. For every win = +1

   2. For every loss = -10

   3. For every missed winning move = -100
